## c9.io

1. Fork this repo
2. Connect c9.io to your GH account
2. Go to the [c9.io repos panel](https://c9.io/account/repos) and click "clone to edit"
3. `> nvm install && npm install`
4. `> npm start`
5. Open http://[REPO_NAME]-[C9_USERNAME].c9users.io in another tab
